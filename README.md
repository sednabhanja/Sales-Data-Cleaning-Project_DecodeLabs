# Data Cleaning & Preparation

## Project Overview
This project focuses on cleaning a raw dataset using Python and Pandas in Google Colab. The goal is to improve data quality by handling missing values, removing duplicate records, correcting data types, and preparing the dataset for analysis.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy

## Data Cleaning Steps
- Imported the dataset
- Explored the dataset
- Identified and handled missing values
- Removed duplicate rows
- Corrected data types
- Verified the cleaned dataset
- Exported the cleaned dataset

## Files
- raw_dataset.xlsx
- cleaned_dataset.csv
- Project1_Data_Cleaning.ipynb

## Outcome
The dataset is now clean, consistent, and ready for further analysis.
